Get started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the app
   ```bash
   pnpm start
   ```

Run Options
In the output, you'll find options to open the app in:

- Development build
- Android emulator
- iOS simulator
- Expo Go

Available Scripts

- `pnpm start`: Start Expo development server
- `pnpm storybook`: Launch Storybook for component development
- `pnpm android`: Start app on Android
- `pnpm ios`: Start app on iOS
- `pnpm web`: Start web version of the app
- `pnpm test`: Run Jest tests
- `pnpm test:watch`: Run tests in watch mode
- `pnpm lint`: Run linter

Development
You can start developing by editing files inside the app directory. This project uses file-based routing.

Testing
Run tests using Jest:

```bash
pnpm test
```

Watch mode:

```bash
pnpm test:watch
```

Storybook
Run Storybook to develop and test components in isolation:

```bash
pnpm storybook
```

Access Storybook at:

- Local: http://localhost:[port]
- Network: http://[your-ip]:[port]

Project Refresh
When you're ready for a clean slate:

```bash
pnpm run reset-project
```
